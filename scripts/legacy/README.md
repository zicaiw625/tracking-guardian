# Legacy Scripts

此目录包含历史版本的验证脚本，仅适用于 v1.0 零订单版本。

## 脚本说明

### verify-zero-pii-v1.sh
- **版本**: v1.0
- **用途**: 验证代码库是否完全移除所有 PCD/PII 相关代码
- **状态**: 已弃用 - 当前产品方向已包含 `read_orders` scope 和订单相关功能
- **注意**: 此脚本会将 `read_orders` 和订单相关 webhooks 视为错误，但当前产品需要这些权限

### verify-pii-cleanup-v1.sh
- **版本**: v1.0
- **用途**: 验证 PII/PCD 清理是否完成
- **状态**: 已弃用 - 当前产品方向已包含订单数据读取功能
- **注意**: 此脚本会将订单相关配置视为错误，但当前产品需要这些功能

## 当前产品方向

当前产品（v1.0+）包含以下功能，需要 `read_orders` scope：
- Reorder 功能（需要 PCD 批准）
- 订单追踪功能
- 对账功能

因此，这些脚本不再适用于当前产品方向。

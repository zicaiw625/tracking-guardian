## 完整描述

### 概述
Checkout 升级助手是一款专为 Shopify 商家设计的迁移工具，帮助您从旧的结账系统平滑迁移到新的 Checkout Extensibility 体系。我们提供一键扫描、智能迁移、实时监控和验收对账，确保您的追踪脚本在升级后正常工作。

### 核心功能

#### 1. 一键扫描与风险报告（免费）
- **自动扫描**：通过 Shopify API 自动检测已安装的 ScriptTags 和 Web Pixels
- **手动分析**：支持手动粘贴 Additional Scripts 内容进行深度分析
- **平台识别**：自动识别 Google Analytics、Meta Pixel、TikTok Pixel 等常见追踪平台
- **风险评分**：基于检测结果生成 0-100 的风险评分和迁移建议
- **截止日期提醒**：根据店铺类型（Plus/非 Plus）显示准确的迁移截止日期

#### 2. 像素迁移中心（付费）
- **多平台支持**：支持 GA4、Meta Pixel、TikTok Pixel、Pinterest Tag
- **向导式配置**：分步骤引导完成像素配置（选择平台 → 填写凭证 → 事件映射 → 测试 → 上线）
- **环境切换**：支持 Test/Live 环境切换，安全测试后再上线
- **配置回滚**：支持配置版本管理和一键回滚
- **事件映射**：可视化配置 Shopify 事件到各平台事件的映射关系

#### 3. Thank you / Order status 页面模块库（付费）
- **订单追踪**：展示物流状态，提升客户体验
- **售后问卷**：收集客户反馈，优化产品和服务
- **再购按钮**：一键再次购买，提升复购率
- **帮助中心**：FAQ + 客服入口，减少售后压力
- **追加销售**：在 Thank you 页面展示推荐商品

#### 4. 事件对账与验收（付费）
- **验收向导**：生成测试清单，指导完成关键场景测试
- **实时监控**：实时查看事件触发情况和参数完整性
- **渠道对账**：验证事件参数与订单金额是否一致
- **报告导出**：一键生成 PDF/CSV 验收报告

#### 5. 上线后监控（付费）
- **事件成功率监控**：实时跟踪事件发送成功率
- **缺参率告警**：检测事件参数缺失情况
- **事件量骤降检测**：及时发现追踪中断
- **多渠道告警**：支持邮件、Slack、Telegram 告警

#### 6. Agency 多店支持（高阶套餐）
- **多店工作区**：统一管理多个店铺
- **批量 Audit**：一键对多个店铺运行扫描
- **批量应用模板**：快速复制配置到多个店铺
- **迁移验收报告**：导出专业的迁移报告

### 常见问题

**Q: 迁移会影响现有追踪吗？**
A: 不会。我们支持 Test/Live 环境切换，您可以先在测试环境验证，确认无误后再切换到生产模式。

**Q: 支持哪些广告平台？**
A: 目前支持 Google Analytics 4、Meta (Facebook) Pixel、TikTok Pixel、Pinterest Tag。更多平台正在开发中。

**Q: 需要技术背景吗？**
A: 不需要。我们提供向导式配置，只需填写平台凭证即可完成迁移。

**Q: 数据安全吗？**
A: 所有凭证使用 AES-256-GCM 加密存储，符合 GDPR 要求，支持数据删除请求。
